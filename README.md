# 课程信息管理系统
##  项目简介
本项目可以帮助学校工作人员管理学生的课程信息
##  功能简介
* 主菜单
    - 显示主界面
* 录入课程信息
    - 课程编号
    - 课程名称
    - 上课时间
    - 课程属性
    - 任课教师
    - 助教教师
    - 上课班级
* 显示课程信息  
* 查询课程信息  
    - 按课程编号查询
    - 按课程名称查询
* 添加课程信息
* 统计课程信息
* 删除课程信息
    - 按编号删除
    - 按名称删除
* 修改课程信息
    - 按编号修改
    - 按名称修改
* 退出管理系统
##  功能函数 
* 增加
* 查询
* 删除
